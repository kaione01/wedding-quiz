## 1. 專案概覽

### 技術架構
| 元件 | 說明 |
|------|------|
| 後端 | Python FastAPI + WebSocket（port 8001） |
| 前端 | 純 HTML/CSS/JS（3 個頁面） |
| 反向代理 | Nginx（port 80）→ 分流到 danmaku:8000 / quiz:8001 |
| 外部存取 | Cloudflare Tunnel（trycloudflare.com 免費隧道） |
| 部署機器 | VPS（4 vCPU / 4 GB RAM / 50 GB SSD） |

### 路由規則
```
https://<tunnel-domain>/quiz/*    → port 8001 (quiz)
https://<tunnel-domain>/          → port 8000 (danmaku/彈幕)
```

## 4. 遊戲流程說明

```
[大螢幕顯示 Lobby]
      ↓
[賓客掃 QR 進入 → 輸入暱稱加入]
      ↓
[MC 點「開始遊戲」]
      ↓
[大螢幕顯示題目 + 10 秒倒數]
[玩家手機同步顯示選項並作答]
      ↓
[時間到 → 等 MC 點「繼續」]
      ↓
[大螢幕揭曉正確答案 + 答題分布圖]
      ↓
[MC 點「繼續」→ 顯示排行榜]
      ↓
[MC 點「繼續」→ 下一題]
      ↓
[所有題目結束 → 顯示最終排行榜]
```

## 7. 新增題目格式

### 文字選項題目
```json
{
  "text": "題目文字",
  "options": ["選項A", "選項B", "選項C", "選項D"],
  "correct": 0,
  "time_limit": 15,
  "option_type": "text",
  "image": null,
  "reveal_media": null
}
```

### 圖片選項題目
```json
{
  "text": "題目文字",
  "options": [
    "/quiz/static/uploads/圖片1.jpg",
    "/quiz/static/uploads/圖片2.jpg",
    "/quiz/static/uploads/圖片3.jpg",
    "/quiz/static/uploads/圖片4.jpg"
  ],
  "correct": 2,
  "time_limit": 10,
  "option_type": "image",
  "image": null,
  "reveal_media": null
}
```

## 8. 已知限制 / 注意事項

1. **Cloudflare Tunnel 網址會變**：每次 VPS 重啟後需更新網址，並通知玩家（或重新展示 QR Code）
2. **玩家需要手機網路**：需要能連到 tunnel 域名（一般 4G/5G 都可以，如有 Wi-Fi 更佳）
3. **背景音效**：需要點擊畫面才會啟動音效（瀏覽器安全限制）
4. **Safari 相容性**：建議使用 Chrome / LINE 內建瀏覽器
5. **題目數量**：目前僅 2 題，活動前需補齊所有題目
